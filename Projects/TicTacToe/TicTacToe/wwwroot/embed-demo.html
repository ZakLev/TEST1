üìù How to Embed in Your Website

Method 1: Simple iframe (Recommended)

<iframe 
    src="http://localhost:5149/embed?openInNewTab=false" 
    width="100%" 
    height="800" 
    frameborder="0"
    style="border: 2px solid #667eea; border-radius: 8px;"
></iframe>

Method 2: For external .com websites

<iframe 
    src="https://your-deployed-app.com/embed?openInNewTab=false" 
    width="100%" 
    height="800" 
    frameborder="0"
    allow="fullscreen"
></iframe>

URL Parameters:

  • openInNewTab=false: Embed games within the iframe (default)
  • openInNewTab=true: Open games in new browser tab

Deployment Options

Option 1: Embedded Mode (openInNewTab=false)

The game embeds directly in the iframe. Good for:

  • Integrated user experience
  • Keeping users on your site
  • When you want to control the layout
  • Single-page applications

Option 2: New Tab Mode (openInNewTab=true)

Games open in a new browser tab. Good for:

  • Keeping your main page clean
  • Full-screen game experience
  • Multiple simultaneous games
  • Better performance for complex pages

üöÄ Deploying to Production

  1. Deploy your Blazor TicTacToe app to a hosting service (Azure, AWS, etc.)
  2. Update the iframe src to point to your deployed URL
  3. Ensure CORS is enabled if embedding cross-domain
  4. Test the embedded functionality thoroughly
